Network Security Scanning

Basic Host Discovery: Employing nmap -v to perform initial host discovery with verbose output, ensuring comprehensive visibility into the scanning process. This step is fundamental for identifying active devices within a target network. Stealth Port Scanning: Utilizing nmap -sS to conduct SYN scans. This technique is critical for efficient and less intrusive port discovery, allowing for the identification of open ports without completing the full TCP handshake, thereby minimizing detection risks. Targeted Port Enumeration: Executing nmap -p to scan specific, pre-defined ports (e.g., 22, 23, 80, 139). This precision scanning is vital for focused assessments of known service ports and optimizing scan efficiency.

Packet Capturing Using Wireshark

Packet capturing with Wireshark is a core skill in cybersecurity. Wireshark is a free, open-source network protocol analyzer that allows professionals to "listen in" on network traffic. It works by putting a computer's network interface into "promiscuous mode," enabling it to capture all data packets passing through that network segment. For cybersecurity, Wireshark is invaluable. It helps in incident response by letting analysts reconstruct attack chains, identify malicious communications (like command-and-control traffic), and gather forensic evidence. For threat hunting, it reveals anomalous network behavior, such as unusual traffic patterns or unauthorized port scans, signaling potential compromises.
